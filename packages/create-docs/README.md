# create-docs

An easy way to generate a static site

## Usage

With npm:

```bash
$ npm create @quick-start/docs
```

With yarn:

```bash
$ yarn create @quick-start/docs
```

With pnpm:

```bash
$ pnpm create @quick-start/docs
```

Then follow the prompts!
